**Depends on:** Phase 8 (monitoring-production-view-service)

## Summary

Implement the console views for order-status-count monitoring, per-sample stock/Depleted-InStock labeling, and the production-line 'in production' head plus FIFO tail display, plus the controller that calls MonitoringService/ProductionLineViewService and hands results to those views. Depends only on phase-8's services; independent of Sample UI, Order UI, and DataMonitor/DummyData UI, so it can be implemented in parallel with phases 10, 11, and 13.
